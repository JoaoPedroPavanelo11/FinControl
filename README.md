# FinControl-API
A FinControl API é uma API RESTful desenvolvida para gerenciamento financeiro pessoal.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
O sistema permite que usuários realizem cadastro e autenticação para controlar receitas, despesas e categorias financeiras, além de visualizar relatórios e saldo total.

O projeto foi desenvolvido com foco em:

-arquitetura backend moderna
-segurança
-organização de código
-boas práticas utilizando NodeJS

------------------------------------------------------------------------------------------------------
Objetivo do Projeto

O objetivo é criar um sistema backend completo simulando uma aplicação real utilizada no mercado.

O projeto demonstra conhecimentos em:

-APIs REST
-autenticação JWT
-banco de dados NoSQL
-arquitetura backend
-segurança
-organização de código
-relacionamentos no MongoDB
-validações
-deploy de aplicações

------------------------------------------------------------------------------------------------------
Funcionalidades do Sistema

-Autenticação de Usuário
-Cadastro de usuários
-Login
-Criptografia de senha
-Geração de token JWT
-Rotas protegidas
----------------------------------------------------------------------------------------------------

Gerenciamento Financeiro
Receitas

O usuário poderá:

-criar receitas
-listar receitas
-editar receitas
-excluir receitas

  Exemplos:
-salário
-freelas
-vendas
----------------------------------------------------------------------------------------------------

Despesas

O usuário poderá:

criar despesas
listar despesas
editar despesas
excluir despesas
Exemplos:
aluguel
alimentação
academia
Categorias

O sistema permitirá:

criação de categorias
separação por tipo
organização financeira
Exemplos:
Alimentação
Transporte
Estudos
Lazer
Dashboard Financeiro

O sistema calculará automaticamente:

total de receitas
total de despesas
saldo disponível
